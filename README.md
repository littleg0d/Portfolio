# Portfolio

This is my personal portfolio website built with React, TypeScript, and Tailwind CSS.

## Features

- **Hero Section**: Introduction and brief overview.
- **About Section**: Information about me.
- **Skills**: Display of technical skills.
- **Projects**: Showcase of my work with details.
- **Contact**: Form and social links to get in touch.

## Tech Stack

- React
- TypeScript
- Tailwind CSS
- Vite

## Getting Started

1. Clone the repository.
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`
